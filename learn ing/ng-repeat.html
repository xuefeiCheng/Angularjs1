ng-repeat指令用于循环输出指定次数的html元素。集合必须是数组或对象。

  • {{x}}

ng-repeat指令不允许id值相同，会报错。为了规避这个错误，需要定义自己的track by表达式。

// 业务上自己生成唯一的id
1. item in items track by item.id
//或者直接拿循环的索引变量$index来用
2. item in items track by $index
另外：如果集合是Javascript对象类型数据，那么就算值一模一样，ng-repeat也不会认为这是相同的对象

  • {{x}}
name country
{{y.Name}} {{y.Country}}
{{key}} {{value}}
